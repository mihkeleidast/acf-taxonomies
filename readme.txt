== Description ==

Choose taxonomies that are attached to a post type.

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-taxonomies` folder into your `wp-content/plugins` folder
2. Activate the Taxonomies plugin via the plugins admin page
3. Create a new field via ACF and select the Taxonomies type
4. Please refer to the Choose taxonomies that are attached to a post type. for more info regarding the field type settings
